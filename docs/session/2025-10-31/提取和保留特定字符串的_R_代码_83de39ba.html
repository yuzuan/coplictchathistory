## 方案 1：使用 sub() 提取最后的数字部分

pd$id <- sub(".*_([0-9_]+)$", "\\1", pd$title)

## 关键说明

1. **sub(".*_([0-9_]+)$", "\\1", pd$title)**：
- .*_ 匹配任意字符直到最后一个下划线
- ([0-9_]+) 捕获数字和下划线的组合
- $ 确保匹配到字符串末尾
- \\1 返回捕获的组

2. **%in% c("132_1", "143")**：检查 ID 是否在指定列表中

3. **边界情况**：
- 如果 ID 可能包含字母，用 [^_]+ 代替 [0-9_]+
- 如果没有下划线，会返回整个字符串
