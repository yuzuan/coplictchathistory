## 使用步骤：

1. **安装必要的包**（首次使用时）：

install.packages("readxl")
install.packages("writexl")
install.packages("BiocManager")
BiocManager::install("org.Mm.eg.db")
BiocManager::install("AnnotationDbi")

2. **修改文件路径**：
- 将脚本中的 "your_file.xlsx" 替换为你的实际文件路径

3. **运行脚本**

## 主要功能：

- **方法1**（推荐）：使用 org.Mm.eg.db 包进行离线转换，速度快
- **方法2**（可选）：使用 biomaRt 包在线转换，更灵活但需要网络

脚本会：
- 读取你的XLSX文件
- 将Gene列中的基因符号转换为UniProt Accession ID
- 添加新列 Protein_Accession
- 显示转换成功率
- 保存结果到新的XLSX文件
- 列出未能转换的基因（如有）
