二、核心生物学发现
1. Tumor 微环境重编程巨噬细胞
Lipid/Ribo TAM (OR=18.2) 和 Mg-Inflam TAM (OR=12.5) 是肿瘤最富集的两种 TAM：

Lipid/Ribo TAM：

Top markers: GAPDH, LGALS1, VIM, S100A9, FTL, SPP1
最高 Lipid-TAM (1.16) + Mo-Identity (0.70) 评分
生物学意义：代谢重编程型 TAM，高糖酵解 (GAPDH)、脂质代谢活跃、S100A8/A9 高表达提示促炎但被肿瘤微环境驯化。这类 TAM 通过脂质摄取获得促肿瘤表型（类似 TREM2+ 脂质相关 TAM），可能通过分泌 SPP1 促进肿瘤侵袭
Mg-Inflam TAM：

Top markers: LYZ, HLA-DRA, SOD2, CST3, CTSS, CD163
最高 Complement-IS (1.21) + Mo-Identity (0.41) 评分
生物学意义：具有抗原呈递能力 (HLA-DR 高) 但同时表达 CD163（M2 marker），提示这是一群被肿瘤微环境"劫持"的炎症巨噬细胞——尽管保留了一定的免疫激活能力，但已开始向免疫抑制方向分化
2. Peripheral 富集的稳态巨噬细胞
SPP1+ Scavenger TAM (OR=0.06) 和 Microglia TAM (OR=0.09) 几乎完全存在于瘤旁组织：

Microglia TAM：

Top markers: DOCK4, PLXDC2, LRMDA, MEF2A
最高 Mg-Identity (0.52) 评分
生物学意义：这是脑驻留巨噬细胞（小胶质细胞）的标志性亚群。在胶质瘤中，小胶质细胞主要存在于瘤旁正常脑组织，维持脑稳态功能。肿瘤核心区几乎不存在 (OR=0.09) 说明肿瘤微环境对原位小胶质细胞有排斥或重编程作用
SPP1+ Scavenger TAM：

Top markers: PDK4, CEBPD, FOS, KLF2, RGS1
最高 Mg-Inflam (1.62) + Mg-Identity (0.53)
生物学意义：虽然命名含 SPP1，但其 top marker 以转录因子和信号调节为主 (CEBPD, KLF2, FOS)，具有组织稳态维护功能。这类细胞在 Peripheral 富集说明它们可能是正常脑组织中的免疫监视细胞
3. 应激反应巨噬细胞的普遍性
HS-UPR TAM（11.3%）和 Systemic-Inflam TAM（23.0%）在两种组织中分布相对均匀：

HS-UPR TAM：热休克蛋白 (HSPA1A/B, HSP90AA1) 显著高表达，HS-UPR 评分 1.44 为全亚群最高。这可能反映手术/组织处理过程中的应激反应（技术性 artifact），也可能代表肿瘤微环境中的持续蛋白折叠压力

Systemic-Inflam TAM：作为最大亚群 (23.0%)，markers 包含 SPP1, TREM2, APOE, CCL3/4，指向经典的疾病相关巨噬细胞 (Disease-Associated Macrophages, DAM) 表型

### 三、关键问题：轨迹方向可能反转

**预期假说**: Microglia TAM (root, ~0) → Mg-Inflam TAM (~1)

**实际结果**: Root cell 在 Microglia TAM 中被选中（pseudotime=0），但 **Microglia TAM 整体中位数 = 0.749**（接近末端），而 Mg-Inflam TAM 中位数 = 0.276（较早）。

**原因分析**：
1. Root 选择使用了 dc1[microglia_mask].idxmax()，选中了 Microglia 中 DC1 值最大的**单个边界细胞**
2. Terminal 选择用 dc1[mask].idxmin()（方向相反），结果 terminal cells 反而被放在低 pseudotime 区域
3. 选中的 root cell 恰好位于 Microglia 集群的边缘（靠近其他亚群），而非集群中心
4. Entropy 佐证：Microglia TAM entropy 最低 (0.639) — 表明它是**终端分化状态**而非起始状态；Complement-IS TAM entropy 最高 (1.276) 才像"祖细胞"

**影响**：当前 pseudotime 排序**不支持** Microglia → Mg-Inflam 的线性分化假说。需要重新选择 root cell 或反转 DC1 方向。
