这是 R 包的 **lazy-load 数据库**文件，是 R CMD INSTALL 时自动从源码的 .rda 文件编译生成的。

该目录下有三个配套文件：

| 文件 | 作用 |
|------|------|
| Rdata.rdb | 数据库主体（序列化的 R 对象，二进制） |
| Rdata.rds | 元数据（记录有哪些对象） |
| Rdata.rdx | 索引（对象在 rdb 中的偏移量和长度） |

当你在 R 中执行 data(anno_obj) 时，R 通过 .rdx 索引找到 anno_obj 在 .rdb 中的位置，按需加载（lazy-load），而不是一次性读入所有数据。

**不能直接用 load() 或 save() 操作这些文件**。它们只能通过 R CMD INSTALL 从源码的 /data/*.rda 重新生成。这也是你之前手动往安装目录加 GPL 后、重装包会丢失的原因。
