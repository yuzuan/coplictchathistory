如何打开工作区

Copilot 2025/05/12 22:46

在VS Code中打开工作区有几种方法：

1. **从VS Code启动界面打开文件夹**：
- 打开VS Code
- 点击左侧的"打开文件夹"按钮
- 或者使用菜单"文件 > 打开文件夹"
- 选择您想要作为工作区的文件夹

2. **使用命令行打开**：
您可以在终端中使用以下命令打开VS Code并指定工作区文件夹：

   code /path/to/your/folder

例如，如果您想打开桌面上的项目文件夹：

   code ~/Desktop/myproject

3. **打开最近使用的文件夹**：
- 在VS Code中点击"文件 > 打开最近的文件夹"
- 选择您之前打开过的文件夹

4. **创建新的工作区并添加文件夹**：
- 打开VS Code
- 点击"文件 > 将文件夹添加到工作区"
- 选择您想要添加的文件夹
- 完成后可以保存工作区为.code-workspace文件

当您打开工作区后，文件资源管理器面板会显示工作区中的所有文件和文件夹，您就可以开始编辑文件或运行代码了。
