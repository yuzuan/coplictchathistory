您可以通过在 theme() 中添加 axis.text.y 来增大通路名称的字体大小：

p_go_dot <- dotplot(ego_all, 
                    showCategory = 10, 
                    split = "ONTOLOGY",
                    font.size = 10) +
    facet_grid(ONTOLOGY ~ ., scales = "free") +
    scale_x_continuous(breaks = c(0.06, 0.12, 0.18)) +
    labs(title = "GO Enrichment Analysis") +
    theme(plot.title = element_text(hjust = 0.5),
          axis.text.y = element_text(size = 12))  # 增大Y轴通路名称字体

print(p_go_dot)
ggsave(file.path(out_dir, "GO_dotplot.pdf"), p_go_dot, width = 6, height = 12)

如果您还想同时调整其他字体，可以添加：

theme(plot.title = element_text(hjust = 0.5),
      axis.text.y = element_text(size = 12),      # Y轴通路名称
      axis.text.x = element_text(size = 11),      # X轴刻度
      axis.title = element_text(size = 12),       # 坐标轴标题
      strip.text = element_text(size = 11))       # 分面标签(BP/CC/MF)

根据需要调整具体的 size 值（例如 12、13、14 等）。

您可以在 facet_grid() 中添加 space = "free_y" 参数，这样每个分面的高度会根据其包含的类别数量自动调整，BP 分面如果类别较多就会相应变小：

p_go_bar <- barplot(ego_all, 
                    x = "Count",
                    showCategory = 10, 
                    split = "ONTOLOGY",
                    font.size = 15) +
    facet_grid(ONTOLOGY ~ ., scales = "free", space = "free_y") +
    labs(title = "GO Enrichment Analysis") +
    theme(plot.title = element_text(hjust = 0.5))

print(p_go_bar)
ggsave(file.path(out_dir, "GO_barplot.pdf"), p_go_bar, width = 10, height = 10)

space = "free_y" 会让每个分面的垂直空间根据其实际的类别数量成比例分配，这样 BP 如果有更多条目就会自动占用更紧凑的空间。

如果您想进一步减小分面之间的间距，可以添加：

theme(plot.title = element_text(hjust = 0.5),
      panel.spacing = unit(0.5, "lines"))  # 调整分面间距

错误原因是 fontface = "bazole_S" 不是一个有效的字体样式。

fontface 参数只接受以下有效值：
- "plain" - 普通字体
- "bold" - 粗体
- "italic" - 斜体
- "bold.italic" - 粗斜体

您的代码中写的是 "bazole_S"（看起来像是误输入或复制粘贴错误），应该改为：

p_heatmap <- Heatmap(
    exp_heatmap_scaled,
    col = heatmap_colors,
    name = "Z-score",
    heatmap_legend_param = list(
        direction = "horizontal",
        title_position = "topcenter",
        legend_width = unit(6, "cm")
    ),
    show_row_names = TRUE,
    show_column_names = FALSE,
    show_row_dend = FALSE,
    show_column_dend = FALSE,
    top_annotation = ha_deg,
    left_annotation = ra_deg,
    column_title_side = "top",
    column_split = annotation_col$Group,
    row_split = annotation_row$Regulation,
    row_names_gp = gpar(fontsize = 6),
    column_title_gp = gpar(fontsize = 12, fontface = "bold"),  # 改为 "bold"
    row_title_gp = gpar(fontsize = 12, fontface = "bold"),     # 改为 "bold"
    cluster_rows = TRUE,
    cluster_columns = TRUE,
    border = TRUE
)

这样就能正常运行了。
