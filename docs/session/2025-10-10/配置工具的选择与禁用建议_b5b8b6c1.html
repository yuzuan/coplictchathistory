- **GitHub 相关**:
- mcp_github_* 所有 GitHub 工具 (如果不频繁操作 GitHub)
- 包括: Issues, Pull Requests, Gists, Discussions, Workflows 等

- fetch_webpage (网页抓取)
- github_repo (GitHub 仓库搜索)
- open_simple_browser (浏览器预览)

## 📝 如何禁用这些工具

您需要在 **VS Code 设置**中调整：

1. **打开设置**: Cmd+, (macOS)
2. **搜索**: "GitHub Copilot Tools"
3. **找到**: github.copilot.chat.tools
4. **取消勾选不需要的工具**

或者在 **设置 JSON** 中配置：

{
  "github.copilot.chat.tools": {
    "github": false,  // 禁用所有 GitHub 工具
    "python": true,   // 保留 Python 工具
    "notebook": false // 禁用 Notebook 工具（如果不用）
  }
}
