data <- fold(
  data = data, k = 4,
  cat_col = 'diagnosis',    # 按diagnosis列进行分层抽样
  id_col = 'participant'    # 确保同一参与者不会同时出现在训练集和测试集

这里创建了4折交叉验证，确保每个折中diagnosis的分布保持平衡。

### 3. 第一个交叉验证（回归问题）

CV1 <- cross_validate(
  data = data,
  formulas = "score ~ diagnosis",  # 用diagnosis预测score
  fold_cols = '.folds',
  family = 'gaussian',             # 高斯分布（连续变量）
  REML = FALSE
)

**结果展示包括：**
- 模型性能指标
- 模型系数
- 训练过程信息

### 4. 第二个交叉验证（分类问题）

CV2 <- cross_validate(
  data = data,
  formulas = "diagnosis~score",    # 用score预测diagnosis
  fold_cols = '.folds',
  family = 'binomial'              # 二项分布（二分类问题）
)

**分类结果包括：**
- **Balanced Accuracy**: 0.736 - 平衡准确率

### 5. 结果可视化

绘制混淆矩阵图，直观显示分类性能。

1. **交叉验证**: 将数据分成K折，轮流用K-1折训练，1折测试
2. **分层抽样**: 确保每折中类别分布均匀
3. **回归vs分类**:
- CV1: 预测连续值（score）
- CV2: 预测类别（diagnosis）
4. **性能评估**: 使用多种指标全面评估模型性能
