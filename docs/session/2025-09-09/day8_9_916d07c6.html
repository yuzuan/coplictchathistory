VSCode安装 MCP 的位置在哪里

Copilot 2025/09/10 22:30

VSCode 插件（如 MCP 相关插件）安装后，默认存放在本地用户目录下的 .vscode/extensions 文件夹中。具体路径如下：

- macOS/Linux: ~/.vscode/extensions/
- Windows: C:\Users\<你的用户名>\.vscode\extensions\

你可以在该目录下查找已安装的所有 VSCode 扩展，包括与 MCP 相关的插件。
如果你想在 VSCode 内部管理或卸载 MCP 插件，可以通过侧边栏的“扩展”视图（快捷键：Cmd+Shift+X）进行搜索、安装和管理。
