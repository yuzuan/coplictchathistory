错误在于 fontface 参数的值不正确。"bsiZNF354A" 不是有效的字体样式，应该使用 "bold"（粗体）。

正确的代码应该是：

p_heatmap <- Heatmap(
    exp_heatmap_scaled,
    col = heatmap_colors,
    name = "Z-score",
    heatmap_legend_param = list(
        direction = "horizontal",
        title_position = "topcenter",
        legend_width = unit(6, "cm")
    ),
    show_row_names = TRUE,
    show_column_names = FALSE,
    show_row_dend = FALSE,
    show_column_dend = FALSE,
    top_annotation = ha_deg,
    left_annotation = ra_deg,
    column_title_side = "top",
    column_split = annotation_col$Group,
    row_split = annotation_row$Regulation,
    row_names_gp = gpar(fontsize = 6),
    column_title_gp = gpar(fontsize = 12, fontface = "bold"),  # 修改这里
    row_title_gp = gpar(fontsize = 12, fontface = "bold"),     # 修改这里
    cluster_rows = TRUE,
    cluster_columns = TRUE,
    border = TRUE
)

将 fontface = "bsiZNF354A" 改为 fontface = "bold" 即可。

有效的 fontface 值包括：
- "plain" - 普通
- "bold" - 粗体
- "italic" - 斜体
- "bold.italic" - 粗斜体
