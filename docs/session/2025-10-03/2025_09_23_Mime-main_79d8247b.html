如何在 Claude code 添加 api

Copilot 2025/10/04 21:16

在 Claude code（如你本地 ~/.claude/config.json）中添加 API，只需将你的 API Key 填写到 config.json 文件的 "api_key" 字段即可。例如：

{
  "model": "claude-3-opus-20240229",
  "temperature": 0.2,
  "max_tokens": 4096,
  "api_key": "你的Claude API Key"
}

保存后，Claude code 工具会自动读取 config.json 并使用你的 API Key 进行调用。无需在代码中硬编码密钥，只需保证 config.json 配置正确即可。

如有具体的 Claude code SDK 或命令行工具，通常会自动读取 ~/.claude/config.json，无需额外操作。
