## Stagewise 插件详细介绍

Stagewise是一个创新的Visual Studio Code插件，它提供了一种名为"Visual vibe coding"（视觉化编码体验）的开发方式。这个工具的主要功能是通过浏览器工具栏将您的前端UI与代码编辑器中的AI代理连接起来，让开发过程更加直观和高效。

### 核心功能

1. **浏览器与编辑器的无缝连接**：
- 在浏览器中选择任何前端元素
- 对选定元素添加评论
- 让AI代理（如GitHub Copilot或Cursor）处理相关代码修改

2. **简化上下文传递**：
- 无需手动复制粘贴文件路径到提示中
- 自动提供AI实时的、基于浏览器的上下文信息

3. **易于使用**：
- 开箱即用，设置简单
- 通过插件系统可自定义和扩展功能

4. **多框架支持**：
- 支持React、Vue、Svelte等多种前端框架
- 提供框架特定的集成包

### 使用方法

1. 安装VS Code插件
2. 在前端项目中安装并注入工具栏：

   pnpm i -D @stagewise/toolbar

3. 在应用中初始化工具栏：

   import { initToolbar } from '@stagewise/toolbar';
   
   const stagewiseConfig = {
     plugins: [],
   };
   
   function setupStagewise() {
     if (process.env.NODE_ENV === 'development') {
       initToolbar(stagewiseConfig);
     }
   }
   
   setupStagewise();

### 支持的AI代理

Stagewise可与多种AI编码助手协同工作：
- ✅ Cursor
- ✅ GitHub Copilot
- ✅ Windsurf
- ✅ Cline
- ✅ Roo-code

### 适用场景

这个插件特别适合前端开发者，能够显著提高以下工作场景的效率：
- 在复杂的前端代码库中进行修改
- 通过可视化方式理解和修改UI组件
- 让AI更准确地理解您想要修改的前端元素
- 在开发过程中快速实现设计变更

### 许可和社区

- 项目基于AGPLv3许可证开发
- 有活跃的社区和Discord支持
- 开源项目，欢迎贡献

总结来说，Stagewise是一个桥接前端UI与代码编辑的创新工具，通过将可视化界面元素直接与代码编辑器中的AI助手连接，使得开发者能够更直观、更高效地进行前端开发工作。
