Copilot 2025/09/16 16:02

你的报错原因是：在 pip install 命令后面加了中文注释 #不更新时会出个warning，而直接运行 pip install 时，注释会被当成包名解析，导致 Invalid requirement 错误。

解决方法：不要把注释和 pip 命令写在同一行，或者用分号隔开。推荐如下：

pip install ipywidgets --upgrade
# 不更新时会出个warning

或者直接删除注释，只保留命令即可。
