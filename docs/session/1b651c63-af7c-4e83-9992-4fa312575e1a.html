现在用 Perplexity 搜索补充更多已知的含 PFS 的 GBM GEO 数据集：

通过 NCBI E-utilities API 和 GEO 网页搜索，共找到 **25 个 GEO Series** 直接匹配 glioblastoma + "progression-free survival"，其中 **16 个数据集的摘要/标题中明确包含 PFS 关键词**。以下是按实用性整理的核心结果：

---

## 含明确 PFS 数据的 GBM GEO 数据集（人类，推荐优先）

| GSE ID | 样本数 | 数据类型 | 描述 |
|--------|--------|----------|------|
| **GSE74187** | 60 | 基因表达芯片 (GPL6480) | **标题直接写明 "OS and PFS"**。60例 GBM 手术样本，含 OS 和 PFS 临床信息，用于生存预测基因签名开发。**最直接的 PFS 数据集** |
| **GSE84010** | 349 | 基因表达芯片 | **AVAglio III 期临床试验**（新诊断 GBM）。349 例，Bevacizumab + RT/TMZ，文中明确 "prolonged PFS"，含分子亚型分类 |
| **GSE72951** | 112 | 基因表达芯片 (DASL + RNA-seq) | **BELOB 临床试验**（复发 GBM）。Bevacizumab + CCNU，鉴定获益患者 |
| **GSE125255** | 56 | SNP 芯片 (OncoScan) | CNV 与 GBM 生存结局关联，含 PFS 数据 |
| **GSE83537** | 36 | 基因表达芯片 (Illumina) | 空间转录组分析，Notch 通路预后标记，IDH1 野生型 GBM，含 PFS |
| **GSE154041** | 71 | RNA-seq | **REGOMA 试验**（复发 GBM）。Regorafenib 治疗的分子签名，含 PFS |
| **GSE154042** | 72 | miRNA-seq | REGOMA 试验配套 miRNA 数据，含 PFS |
| **GSE283494** | 33 | 甲基化芯片 | 复发 GBM Regorafenib 治疗的甲基化评分预测 PFS |
| **GSE242790** | 30 | scRNA-seq | 新诊断 GBM 的 CAR-T (EGFRvIII) + Pembrolizumab I 期临床试验 |
| **GSE296577** | 20 | 其他 | 溶瘤病毒治疗 GBM 的 T 细胞持续激活，含生存数据 |
| **GSE240704** | 363 | 甲基化芯片 | 高级别胶质瘤神经表观遗传学签名预后分析，独立预测生存 |
| **GSE24072** | 32 | 基因表达芯片 | VAV1 在 GBM 中的表达与预后 |
| **GSE279950** | 109 | 甲基化芯片 | IDH 突变低级别胶质瘤 MGMT 甲基化评分预测 TMZ 获益（低级别为主） |

## 其他注意事项

- **GSE285215/285216/285287/285288**：这 4 个实为同一项 ferroptosis 研究的不同组学数据，**小鼠模型**，不是人类临床数据
- **GSE126168**：仅 7 个样本，RT-PCR 数据，价值较低
- **GSE121719-121723**：SOX10 调控 GBM 分子亚型的多组学数据（甲基化/ChIP-seq/RNA-seq），含 PFS 但主要关注表观遗传学

## 经典大型数据集（补充，非 GEO 直接检索到但常用）

| 数据集 | 来源 | 样本数 | 说明 |
|--------|------|--------|------|
| **TCGA-GBM** | GDC/TCGA | ~600 | 标准化临床数据含 PFS，可从 cBioPortal 获取 |
| **CGGA** (mRNAseq_325/693) | CGGA 官网 | 325/693 | 中国胶质瘤基因组图谱，含 PFS |
| **GSE16011** | GEO | 284 | 经典 GBM 表达谱，含临床生存数据 |
| **GSE4412** | GEO | 85 | GBM 表达谱 + 生存 |
| **GSE13041** | GEO | 267 | ANOCEF trial，新诊断 GBM，含临床特征 |
