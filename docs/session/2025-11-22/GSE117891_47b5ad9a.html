文件夹内的数据类型是 **gzip 压缩的 TSV (Tab-Separated Values) 文件**。

每个文件（例如 GSM3314532_S1P1.UMI_count.tsv.gz）都包含了一个特定样本的单细胞基因表达矩阵（UMI counts），其中：
* **行 (Rows)**：通常是基因名称。
* **列 (Columns)**：通常是细胞条形码 (Cell Barcodes)。
* **值 (Values)**：UMI 计数，代表基因表达量。

### 如何导入数据

你可以使用 Python 的 pandas 库来读取这些文件。由于文件是压缩的，需要指定 compression='gzip' 参数。

#### 1. 读取单个文件示例

import pandas as pd

# 指定文件路径
file_path = "GSE117891_RAW/GSM3314532_S1P1.UMI_count.tsv.gz"

# 读取数据
# sep='\t': 指定分隔符为制表符
# compression='gzip': 指定解压方式
# index_col=0: 将第一列（基因名）作为索引
df = pd.read_csv(file_path, sep='\t', compression='gzip', index_col=0)

# 查看前几行
print(df.head())
