针对您的需求，推荐使用 pandas 的 chunksize 参数分批读取大文件，并逐步拼接、合并，最后再做质控和AnnData对象创建。这样可以有效避免内存溢出。下面是分块读取的优化代码：

import pandas as pd
import numpy as np
import gc

# 设置质控参数
min_cells_per_gene = 20
min_genes_per_cell = 200

data_file = "GSE154795_Processed_data.txt.gz"
print(f"\n分块读取数据文件: {data_file}")

chunksize = 2000  # 每次读取2000行基因，可根据内存调整
chunks = []
total_genes = 0

for chunk in pd.read_csv(data_file, sep=',', compression='gzip', header=0, index_col=0, dtype=np.float32, chunksize=chunksize):
    print(f"读取分块: {chunk.shape}")
    # 质控过滤（每块先过滤基因表达细胞数）
    genes_pass_filter = (chunk != 0).sum(axis=1) >= min_cells_per_gene
    chunk_filtered = chunk.loc[genes_pass_filter]
    chunks.append(chunk_filtered)
    total_genes += chunk_filtered.shape[0]
    gc.collect()

print(f"分块拼接后总基因数: {total_genes}")
df = pd.concat(chunks)
del chunks
gc.collect()

# 细胞过滤
cells_pass_filter = (df != 0).sum(axis=0) >= min_genes_per_cell
df_filtered = df.loc[:, cells_pass_filter]
print(f"最终过滤后数据维度: {df_filtered.shape}")

# 转置为scanpy格式
df_t = df_filtered.T

import scanpy as sc
adata = sc.AnnData(df_t.values)
adata.obs_names = df_filtered.columns
adata.var_names = df_filtered.index

print(f"\nAnnData对象创建完成: {adata.shape}")
print(f"- 细胞数量: {adata.n_obs:,}")
print(f"- 基因数量: {adata.n_vars:,}")
print("分块读取和质控完成！")

- 用 chunksize 分批读取，避免一次性加载全部数据。
- 每块先做基因过滤，最后拼接所有块，再做细胞过滤。
- 释放不必要的变量及时回收内存。
