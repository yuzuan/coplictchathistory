什么问题

Copilot 2025/09/06 22:34

你的 notebook 报错 ModuleNotFoundError: No module named 'distutils'，这是因为 patchworklib 依赖 distutils，而 Python 3.12 及以上版本已移除 distutils，导致无法导入。

你已经用 conda 安装了 setuptools，这通常会解决 distutils 缺失问题。如果还有报错，说明 patchworklib 依赖的 distutils 代码不兼容新版本 Python。

解决方法：
1. 确认 setuptools 已安装（你已完成）。
2. 如果依然报错，建议：
- 降级 Python 到 3.11 或更低版本（推荐，兼容性最好）。
- 或等待 patchworklib 官方适配 Python 3.12+。
- 临时方案：手动修改 patchworklib 代码，将 from distutils.version import LooseVersion 改为 from setuptools._distutils.version import LooseVersion（不推荐，易出新问题）。

建议：如需长期使用 patchworklib，推荐使用 Python 3.11 环境。你可以用 conda 创建新环境：

conda create -n py311 python=3.11
conda activate py311
conda install matplotlib seaborn plotnine patchworklib
