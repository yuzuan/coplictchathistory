TOTP 密钥是什么

Copilot 2025/10/29 11:37

TOTP（Time-based One-Time Password）密钥是用于生成双因素认证（2FA）验证码的密钥字符串。

## 🔐 什么是 TOTP 密钥？

TOTP 密钥是一个 **base32 编码的字符串**（通常是 16-32 个字符），例如：

JBSWY3DPEHPK3PXP

当你在 Microsoft 账号启用双因素认证（authenticator app）时，微软会给你：
1. 一个 **二维码**（扫描添加到认证器 App）
2. 或一个 **密钥字符串**（手动输入）

这个密钥就是 TOTP secret。
