1. 打开 **Zotero → 首选项（Preferences）→ Zotero MCP Plugin** 标签页
2. **勾选 Enable Server** 启用 MCP 服务器
3. **将端口改为 23121**（因为 23120 已被 ZotLink 占用）
4. 点击保存/确认

完成后回到 VS Code，Cmd+Shift+P → Developer: Reload Window 重新加载即可连接。
