下面按 step1–14 的脚本流程简要解析（基于对应 R 脚本）：

1. step1：读取 4 个 ST 矩阵，创建 Seurat 对象，SCTransform、PCA、聚类、UMAP；为每个 cluster 计算差异基因并保存；统计每个 cluster spot 数量并输出（给 LR 节点属性用）。见

2. step2：把 ST 聚类结果映射到空白背景和 H&E 图像上（含坐标校准）。见

3. step3：加载 scRNA‑seq（Shih 2018）数据做 QC、归一化、降维聚类、标注细胞类型；计算差异基因并做热图。见

4. step4：做 MIA（多模态交集分析），用 scRNA‑seq 与 ST cluster DE 基因的交集显著性，输出每个样本的 MIA 矩阵并画热图。见

5. step5：根据 MIA 结果把 ST cluster 划分为肿瘤/间质，写入 pool_tumor 和 pool_stroma 元数据并保存。见

6. step6：将肿瘤 cluster 映射成二值图并做边缘检测，得到肿瘤轮廓图。见

7. step7：在肿瘤轮廓背景上绘制单基因空间表达（肿瘤/间质 marker），并生成色标。见

8. step8：选择 CAF 相关 cluster，在肿瘤轮廓图上可视化 CAF 空间分布。见

9. step9：汇总各 cluster 的差异基因，做 avg_logFC 热图；同时针对 CAF 相关基因做子集热图。见

10. step10：基于 DAVID 的 GO‑BP 基因集，对每个 cluster 做超几何富集，输出 top GO 热图与选定通路热图。见

11. step11：当前脚本内容与 step10 相同（GO 富集+热图），未见 CAF 特异逻辑。见

12. step12：计算所有间质 spot 到最近肿瘤 spot 的距离分布并绘图，输出均值/标准差。见

13. step13：计算 CAF 基因在间质中的平均表达随距离变化的曲线（距离‑表达关系）。见

14. step14：加载配体‑受体库，寻找肿瘤‑间质界面最近邻 spot 的配体‑受体对，输出相互作用及可视化最近邻位置。见
