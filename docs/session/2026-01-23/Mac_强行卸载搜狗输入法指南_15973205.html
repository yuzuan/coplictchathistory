Mac 如何强行卸载搜狗输入法

Copilot 2026/01/23 15:36

在Mac上强制卸载搜狗输入法，可以按以下步骤操作：

## 方法一：使用官方卸载工具
1. 打开搜狗输入法设置
2. 点击"卸载搜狗输入法"按钮

## 方法二：手动彻底删除

**1. 从系统设置中移除输入法：**
- 打开"系统设置" > "键盘" > "输入法"
- 选中搜狗输入法，点击"-"号删除

**2. 删除应用程序和相关文件：**

可以在终端中执行以下命令来彻底删除：

# 删除应用程序
sudo rm -rf /Library/Input\ Methods/SogouInput.app

# 删除偏好设置文件
rm -rf ~/Library/Preferences/com.sogou.inputmethod.*

# 删除应用支持文件
rm -rf ~/Library/Application\ Support/Sogou*

# 删除缓存文件
rm -rf ~/Library/Caches/com.sogou.inputmethod.*
rm -rf ~/Library/Caches/SogouServices

# 删除其他可能的残留文件
rm -rf ~/Library/Saved\ Application\ State/com.sogou.*
rm -rf ~/Library/Cookies/com.sogou.*

# 重启输入法服务
killall SogouServices 2>/dev/null

**3. 重启电脑**，确保所有更改生效。
