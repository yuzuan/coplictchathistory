## 安装步骤：

### 1. **克隆仓库到本地**
在终端运行：

cd ~/Downloads
git clone https://github.com/luwill/research-skills.git

### 2. **找到 Cline 配置目录**
Cline 插件通常使用以下目录结构：
- macOS: ~/.cline/ 或通过 VS Code 设置
- 也可能通过 VS Code 的设置界面配置

### 3. **创建或找到 skills 目录**

mkdir -p ~/.cline/skills

### 4. **复制技能文件**

cp -r ~/Downloads/research-skills ~/.cline/skills/medical-imaging-review

### 5. **在 Cline 中启用**
在 VS Code 中：
1. 打开 Cline 插件
2. 进入设置（Settings）
3. 查找 "Custom Instructions" 或 "Skills" 配置项
4. 指向 ~/.cline/skills/medical-imaging-review 目录

**注意**：Cline 的具体配置方式可能与 Claude Code 不同。你可以：
- 查看 Cline 的文档了解其自定义技能的配置方式
- 或者将 SKILL.md 的内容作为自定义指令添加到 Cline 的设置中
