**更新后的研究主线**
我现在会把主线改成两层，而不是之前那种单层叙事。

第二层是谱系分叉：
同样是外围起点，有一支更像 microglia-like inflammatory branch，体现在 Mg-Inflam 和真正的 Homeostatic MG；另一支更像 macrophage-like repair/phagocytic branch，体现在 Complement/Scavenger/APOE-Lipid。进入肿瘤核心之后，这两支都可能汇入应激程序，但速度和路径不同。

GBM 髓系细胞并不是沿一条单线从瘤旁走向肿瘤核心，而是从“炎症型外围入口”和“吞噬修复型外围入口”两条支路出发，在核心区汇入以 Scavenger-Lipid 和 Stress-HSUPR 为代表的肿瘤适应程序；三位置数据定义了这条空间主轴，scGBM 和 snGBM 的 0.8 结果则把它拆解成可对齐的细分状态。

3. 把 scGBM 和 snGBM 当细分验证集，用来解释同一 program 在不同谱系里的分叉。
