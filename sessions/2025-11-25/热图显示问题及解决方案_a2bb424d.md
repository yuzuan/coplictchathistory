### 🤖 Copilot <sub>2025/11/25 10:31</sub>

错误很清楚：**`gaps_col` 和 `cluster_cols = TRUE` 不能同时使用**。

`gaps_col` 是用来在指定位置画分隔线的，但如果开启了列聚类（`cluster_cols = TRUE`），样本顺序会被重新排列，分隔线的位置就会乱掉。

### 解决方案

**方案一：关闭列聚类（推荐）**
如果你想保持分组的视觉分隔，应该关闭列聚类：

```r
pheatmap(
  mat_ordered,
  annotation_col = annotation_col,
  annotation_colors = annotation_colors,
  show_rownames = show_names,
  show_colnames = FALSE,
  cluster_rows = TRUE,
  cluster_cols = FALSE,   # 关闭列聚类
  scale = "none",
  color = colorRampPalette(c("blue", "white", "red"))(100),
  border_color = NA,
  gaps_col = cumsum(table(group_ordered))[-length(table(group_ordered))],
  main = paste0(gene_set, " DEGs Heatmap (n=", n_genes, ")"),
  fontsize_row = font_size,
  annotation_names_col = TRUE,
  annotation_legend = TRUE
)
```

**方案二：保留列聚类，去掉分隔线**
如果你更想要聚类效果，就去掉 `gaps_col`：

```r
pheatmap(
  mat_ordered,
  annotation_col = annotation_col,
  annotation_colors = annotation_colors,
  show_rownames = show_names,
  show_colnames = FALSE,
  cluster_rows = TRUE,
  cluster_cols = TRUE,    # 保留列聚类
  scale = "none",
  color = colorRampPalette(c("blue", "white", "red"))(100),
  border_color = NA,
  # 不要 gaps_col
  main = paste0(gene_set, " DEGs Heatmap (n=", n_genes, ")"),
  fontsize_row = font_size,
  annotation_names_col = TRUE,
  annotation_legend = TRUE
)
```

### 推荐方案一

对于差异基因热图，通常**关闭列聚类 + 使用分隔线**更好，因为：
1. 可以清晰看到两组之间的边界
2. 样本已经按分组排序了（`group_order <- order(group)`）
3. 注释条已经显示了分组信息
