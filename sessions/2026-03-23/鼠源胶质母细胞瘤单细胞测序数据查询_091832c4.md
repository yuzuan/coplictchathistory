## Periphery vs Border 区别

在这个 GSE241037 数据集中，三个空间分区代表从肿瘤中心向外的**同心空间梯度**：

```
         ┌─────────────────────────┐
         │ 正常脑组织 (Normal Brain)  │
         │  ┌───────────────────┐  │
         │  │   Periphery       │  │  ← 瘤旁区域：肿瘤外围的脑实质
         │  │  ┌─────────────┐  │  │     远离肿瘤主体，但仍受肿瘤微环境影响
         │  │  │   Border    │  │  │  ← 肿瘤边缘/浸润前沿：肿瘤与正常脑交界处
         │  │  │  ┌───────┐  │  │  │     肿瘤细胞正在主动浸润的区域
         │  │  │  │ Core  │  │  │  │  ← 肿瘤核心：肿瘤主体中心
         │  │  │  │       │  │  │  │     高肿瘤细胞密度、坏死、缺氧
         │  │  │  └───────┘  │  │  │
         │  │  └─────────────┘  │  │
         │  └───────────────────┘  │
         └─────────────────────────┘
```

| 分区 | 中文对应 | 定义 | 关键特征 |
|------|----------|------|----------|
| **Core** | 肿瘤核心 | 肿瘤主体的中心区域 | 高肿瘤细胞密度、坏死区、缺氧、免疫抑制最强 |
| **Border** | 肿瘤边缘 / 浸润前沿 | 肿瘤与正常脑组织的**交界面** | 肿瘤细胞活跃浸润、基质重塑、免疫细胞频繁交互 |
| **Periphery** | **瘤旁组织** | 肿瘤边缘**外侧**的脑实质 | 形态上基本正常，但已有免疫细胞浸润和微环境改变 |

### 与你的研究关系

- **Periphery = 你强调的"瘤旁组织"**：这是肿瘤外围的脑组织，虽然没有明显的肿瘤细胞占据，但免疫微环境已经发生改变。该区域的 TAM 可能代表最早期对肿瘤产生应答的免疫细胞群。

## Periphery vs Border 区别

在这个 GSE241037 数据集中，三个空间分区代表从肿瘤中心向外的**同心空间梯度**：

```
         ┌─────────────────────────┐
         │ 正常脑组织 (Normal Brain)  │
         │  ┌───────────────────┐  │
         │  │   Periphery       │  │  ← 瘤旁区域：肿瘤外围的脑实质
         │  │  ┌─────────────┐  │  │     远离肿瘤主体，但仍受肿瘤微环境影响
         │  │  │   Border    │  │  │  ← 肿瘤边缘/浸润前沿：肿瘤与正常脑交界处
         │  │  │  ┌───────┐  │  │  │     肿瘤细胞正在主动浸润的区域
         │  │  │  │ Core  │  │  │  │  ← 肿瘤核心：肿瘤主体中心
         │  │  │  │       │  │  │  │     高肿瘤细胞密度、坏死、缺氧
         │  │  │  └───────┘  │  │  │
         │  │  └─────────────┘  │  │
         │  └───────────────────┘  │
         └─────────────────────────┘
```

| 分区 | 中文对应 | 定义 | 关键特征 |
|------|----------|------|----------|
| **Core** | 肿瘤核心 | 肿瘤主体的中心区域 | 高肿瘤细胞密度、坏死区、缺氧、免疫抑制最强 |
| **Border** | 肿瘤边缘 / 浸润前沿 | 肿瘤与正常脑组织的**交界面** | 肿瘤细胞活跃浸润、基质重塑、免疫细胞频繁交互 |
| **Periphery** | **瘤旁组织** | 肿瘤边缘**外侧**的脑实质 | 形态上基本正常，但已有免疫细胞浸润和微环境改变 |

### 与你的研究关系

- **Periphery = 你强调的"瘤旁组织"**：这是肿瘤外围的脑组织，虽然没有明显的肿瘤细胞占据，但免疫微环境已经发生改变。该区域的 TAM 可能代表最早期对肿瘤产生应答的免疫细胞群。

- **Border ≠ 瘤旁组织**：Border 仍然是肿瘤的一部分（浸润前沿），有大量肿瘤细胞与免疫细胞的直接混合。是肿瘤主动侵袭的界面。

- **研究价值**：比较 Periphery vs Border vs Core 的免疫细胞组成变化，可以揭示 TAM 从瘤旁到核心的空间演化轨迹。2024 CNS 会议摘要中发现的 **Early Responsive Macrophage (ERMs)** 就富集在早期阶段和空间上更靠外的区域（Periphery）。
