### 🤖 Copilot <sub>2026/02/23 21:48</sub>

**GitHub Copilot 目前不原生支持 Agent Skills 标准**，该 repo 明确支持的工具只有 Cursor、Claude Code 和 Codex CLI。

---

**原因：** Agent Skills 依赖于 AI agent 能自动扫描 `~/.claude/skills/` 或 `~/.cursor/skills/` 目录并加载 `SKILL.md`，GitHub Copilot 没有这个机制。

---

**但在 VS Code + Copilot Chat 中可以手动变通使用：**

**方法一：用 `#file` 引用 SKILL.md 作为上下文**

在 Copilot Chat 输入框中直接引用技能文件：

```
#file:path/to/claude-scientific-skills/scientific-skills/scanpy/SKILL.md

请参考上述文档，帮我用 Scanpy 分析 10X Genomics 单细胞数据，做 QC 和细胞类型鉴定。
```

**方法二：添加为工作区自定义指令**

在 `.github/copilot-instructions.md`（VS Code Copilot 支持读取此文件）中粘贴 SKILL.md 的核心内容，让 Copilot 持续感知这些工具的用法。

**方法三：将技能文件加入工作区并用 `@workspace` 查询**

把需要的 skill 文件夹复制到项目目录，然后：
```
@workspace 用 Scanpy 分析单细胞数据，参考项目中的 scanpy/SKILL.md 进行 QC 和降维聚类。
```

---

**推荐方案：** 如果你主要做单细胞等科研分析，建议切换到 **Claude Code** 或 **Cursor**，可以完整发挥这套 140+ skills 的自动调用能力，无需手动引用。
