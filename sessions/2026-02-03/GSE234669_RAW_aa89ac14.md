### 处理逻辑
1.  **提取样本名**：从文件名中提取 `GSM` 开头的 ID 作为列名（例如 `GSM7473078`）。
2.  **过滤非基因行**：剔除了以 `N_` 开头的统计行（如 `N_unmapped` 等）。
3.  **合并数据**：以 `gene` 列为基准，将所有样本的 counts 值合并。
4.  **保存格式**：保存为制表符分隔的

，首列列名为 `gene`。

### R 代码实现 
你可以直接在 merge_data.R 中查看完整脚本：

```
# 设置工作目录
dir_path <- "/Users/yuzuan/Desktop/fastGEO/test/00_GEO_data_GSE234669/GSE234669_RAW"
setwd(dir_path)

# 获取所有样本文件
files <- list.files(pattern = "\\.txt\\.gz$")
data_list <- list()

for (f in files) {
  # 提取 GSM ID
  gsm_id <- sub("(_.*)$", "", f)
  # 读取数据
  temp_data <- read.table(f, header = TRUE, sep = "\t", stringsAsFactors = FALSE)
  # 过滤掉统计行 (以 N_ 开头的行)
  temp_data <- temp_data[!grepl("^N_", temp_data[[1]]), ]
  # 重命名列：第一列为 gene，第二列为样本 GSM ID
  colnames(temp_data) <- c("gene", gsm_id)
  data_list[[gsm_id]] <- temp_data
}

# 合并所有样本
merged_data <- Reduce(function(x, y) merge(x, y, by = "gene", all = TRUE), data_list)

# 保存为 txt 文件，确保包含 gene 列名
write.table(merged_data, "GSE234669_expression_matrix.txt", sep = "\t", quote = FALSE, row.names = FALSE)
```
