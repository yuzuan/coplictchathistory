```json
{
  "model": "claude-3-opus-20240229",
  "temperature": 0.2,
  "max_tokens": 4096,
  "api_key": "你的Claude API Key"
}
```

保存后，Claude code 工具会自动读取 config.json 并使用你的 API Key 进行调用。无需在代码中硬编码密钥，只需保证 config.json 配置正确即可。
