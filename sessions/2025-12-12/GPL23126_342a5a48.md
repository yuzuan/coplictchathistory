在 VS Code 中切换 GitHub Copilot 账号：

1. **点击左下角的账户图标**（齿轮图标旁边）
2. **选择当前登录的账号** → 点击 **"Sign Out"**
3. **重新点击账户图标** → 选择 **"Sign in with GitHub to use GitHub Copilot"**
4. **使用新的 GitHub 账号登录**

或者通过命令面板：
- 按 `Cmd+Shift+P`（macOS）
- 输入 `GitHub: Sign Out`
- 然后再输入 `GitHub: Sign In` 登录新账号
