明白了，格式是 `NM_xxx // GeneSymbol // description`。现在提取基因名并转换：

```
# 提取第一个基因名（格式：xxx // GeneSymbol // ...）
extract_gene <- function(x) {
  if (x == "---" || is.na(x)) return(NA)
  # 提取第一个 // 后面的基因名
  parts <- strsplit(x, " // ")[[1]]
  if (length(parts) >= 2) return(parts[2])
  return(NA)
}

# 创建探针ID到基因名的映射
annot$gene_symbol <- sapply(annot$gene_assignment, extract_gene)

# 查看结果
```
