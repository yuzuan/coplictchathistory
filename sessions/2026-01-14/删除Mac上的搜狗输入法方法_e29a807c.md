要删除Mac上的搜狗输入法，按以下步骤操作：

## 方法一：使用搜狗自带的卸载工具
1. 打开访达，进入 **应用程序** 文件夹
2. 找到 **搜狗输入法** 文件夹
3. 双击运行 **卸载搜狗输入法** 程序
4. 按提示完成卸载

## 方法二：手动删除
1. **关闭搜狗输入法**：在菜单栏切换到其他输入法

2. **打开系统设置**：
   - 进入 **系统设置** > **键盘** > **输入法**
   - 选中搜狗输入法，点击 **-** 号移除

3. **删除相关文件**（在访达中按 `⌘⇧G` 前往文件夹）：
   ```
   ~/Library/Input Methods/
   ```
   删除 `SogouInput.app`

   ```
   ~/Library/Preferences/
   ```
   删除 `com.sogou.inputmethod.*` 相关文件

   ```
   ~/Library/Application Support/
   ```
   删除 `Sogou` 文件夹

   ```
   ~/Library/Caches/
   ```
   删除 `com.sogou.inputmethod.*` 相关文件

4. **清理启动项**：
   - 进入 **系统设置** > **通用** > **登录项**
   - 移除搜狗相关的启动项

5. **重启电脑** 使更改生效

## 注意事项
- 删除前建议先备份个人词库和设置
- 如遇到权限问题，可能需要输入管理员密码
- 部分系统文件可能需要在恢复模式下删除（较少见）

现在安装的第三方输入法都删除不了是什么原因

### 🤖 Copilot <sub>2026/01/14 15:48</sub>

Mac第三方输入法删除不了通常有以下几个原因和解决方法：

## 常见原因

### 1. **系统完整性保护 (SIP) 限制**
- macOS的安全机制可能阻止删除输入法文件
- 某些输入法文件位于受保护的系统目录

### 2. **输入法进程仍在运行**
- 输入法后台进程没有完全退出
- 系统仍在使用该输入法

### 3. **权限不足**
- 需要管理员权限才能删除
- 文件被锁定或标记为只读

### 4. **残留的启动项或守护进程**
- Launch Agents/Daemons 仍在运行
- 自动重新启动输入法服务

## 解决方法

### 步骤1：强制退出输入法进程
打开 **活动监视器**（应用程序 > 实用工具），搜索输入法相关进程（如"Sogou"、"输入法"等），选中后点击强制退出。

### 步骤2：禁用系统完整性保护（慎重）
如果确实需要，可以临时关闭SIP：
1. 重启Mac，按住 `⌘R` 进入恢复模式
2. 打开终端，输入：
   ```bash
   csrutil disable
   ```
3. 重启后删除输入法
4. 再次进入恢复模式，重新启用SIP：
   ```bash
   csrutil enable
   ```

### 步骤3：使用终端命令删除
```bash
# 删除输入法应用
sudo rm -rf ~/Library/Input\ Methods/输入法名称.app

# 删除配置文件
sudo rm -rf ~/Library/Preferences/com.输入法.*

# 删除缓存
sudo rm -rf ~/Library/Caches/com.输入法.*

# 删除启动项
sudo rm -rf ~/Library/LaunchAgents/com.输入法.*
```

### 步骤4：清理隐藏的守护进程
```bash
# 查找并卸载LaunchAgents
launchctl list | grep 输入法关键词
launchctl remove 对应的服务名
```

### 步骤5：重置输入法设置
```bash
# 重置输入法配置
sudo killall -9 SystemUIServer
sudo killall -9 ControlCenter
```

## 推荐做法
1. 先在 **系统设置 > 键盘 > 输入法** 中移除
2. 重启电脑到**安全模式**（开机时按住Shift键）
3. 在安全模式下删除输入法文件
4. 正常重启

如果以上方法都不行，可能是输入法与系统深度集成，建议联系输入法开发商获取专门的卸载工具。
