### **3-5 条推荐研究主线**

#### **1. 瘤旁→肿瘤核心的"稳态丧失 → 应激适应"轴**（超强证据等级）

**聚焦问题**：Homeostatic_MG (100% 外周) → HS-UPR TAM (86% 肿瘤) 之间是否存在连续的状态转换轨迹？

**推进方向**：
- 用 RNA velocity / pseudotime (monocle3/scVelo) 在整合的三个 dataset 上推断 **Homeostatic → Stress 的转变轨迹**
- 关键检验点：过渡态细胞（中间表型）在 Edge 区域是否富集（支持三位置 GBM 的细致空间）
- 转录因子动态：从 ZFP36L2 (stasis) → ATF4/CHOP (UPR) 的激活时序
- **验证方向**：spatial transcriptomics 确认梯度，或 immunofluorescence 验证关键标记物的空间分布

---

#### **2. 补体-清道夫双重激活的 TAM "中间功能态" 定位**（中等证据）

**聚焦问题**：Complement_Scavenger_TAM 何以跨区域位置相对中立？它代表"M2 样清理功能"还是"M1→M2 的过渡"？

**推进方向**：
- **细胞互作假说**：此 program 是否富集在基质-免疫-肿瘤 niche 的交界？用 CellChat/LIANA 测定其与 CAF/内皮/肿瘤细胞的关键通讯
- **代谢特征**：lipid uptake (LDL/HDL 通路) 与 complement-mediated phagocytosis 的共激活是否指示"补体调理吞噬"
- **位置再检视**：snGBM 99% Peri vs scGBM 无强位置偏好，差异源于细胞类型输入（Microglia+Macrophage 混合？）还是样本异质性？
- **直接实验**：FACS 分离此 program 的细胞，测定补体受体(CR1/CR3/CR4) 与脂质代谢标记的共表达

---

#### **3. scGBM 的 Mg_Inflam TAM 与 snGBM 的完全缺失之谜 → Microglia-Macrophage 分工假说**（强差异等级）

**聚焦问题**：为何 scGBM 中 Mg_Inflam 占 35-40%，而 snGBM (纯 Macrophage) 中完全无对应？

**推进方向**：
- **清晰假说**：Mg_Inflam 程序代表 **Microglia 特异的核心功能**；macrophage 无法复制此高动力 IEG (JUN/FOS/KLF2) 激活
- **验证策略**：
  1. 在三个 dataset 中分离 Microglia vs Macrophage 两个族群，分别对这两者做 Mg_Inflam signature 评分 
  2. 假设：Microglia 高评分 & 肿瘤富集；Macrophage 低评分 or 不存在
- **机制推断**：CX3CR1 (Microglia marker) 与 Mg_Inflam 模块表达的相关性强度
- **临床意义**：Microglia 清除可能选择性消灭 Mg_Inflam 功能，保留 Macrophage-driven 清道功能 → CAR-T 联合 CSF1R 抑制的设计依据

---

#### **4. HS-UPR TAM 的肿瘤核心应激微环境征**（超强证据，已高度稳健）

**聚焦问题**：HS-UPR 完全肿瘤限制（snGBM 86% Tumor），如何与 HIF1A/缺氧轴互联？

**推进方向**：
- **整合验证**：在同样的样本上测 HIF1A activity (e.g., signature score) 与 HS-UPR 评分的相关性
- **上游驱动**：缺氧 vs 营养饥饿 vs 肿瘤细胞分泌物，哪个是主驱动？用 bulk/空间数据反卷积计算 pO2 gradient
- **治疗靶点**：
  - 抑制 HSP90/DNAJB1 是否选择性杀伤应激 TAM（相对于稳态MG安全）
  - 缓解 UPR 是否重启 macrophage 的清道功能（从应激模式重编程到清理模式）
- **预测价值**：HS-UPR score 与免疫检查点反应的关联

#### **5. "外周-中间-核心" 三层次模型的稳健性验证与跨物种推广**（中等优先级，方法学意义）

**聚焦问题**：三位置 GBM 中的 Edge 区域是否真正代表"生物学中间态"，还是仅仅是解剖学过渡？

**推进方向**：
- **三位置 GBM 的优势利用**：其他两个 notebook 缺乏 Edge 的详细分析，这是独特优势

- **跨数据集验证**：
  - 用 scGBM/snGBM 的样本内部人工划分"近肿瘤边界" vs "远离边界" macrophage，检测类似梯度
  - 或用 spatial transcriptomics 数据直接测距离-表达型的连续关系
- **临床转移**：建立"位置特异性 myeloid program 特征"作为诊断/预后标记，用于肿瘤核心穿刺活检样本的推断
