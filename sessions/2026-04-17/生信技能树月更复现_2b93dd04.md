### **4. HIF1A_SCORE & NFKB1_SCORE COMPUTATION METHODS**

**Method:** `sc.tl.score_genes()` (from Scanpy)

#### **Exact Code Pattern** (from [CT39_permutation_test.ipynb](CT39_permutation_test.ipynb) lines 945-947, [cross_project_summary2.ipynb](cross_project_summary2.ipynb) lines 1858-1866):

```python
# HIF1A score (hypoxia/stress genes)
avail_hif = [g for g in hif1a_genes if g in adata.var_names]
sc.tl.score_genes(adata, gene_list=avail_hif, score_name='HIF1A_score', 
                   use_raw=False, ctrl_size=50)

# NFκB score (inflammatory/NF-κB pathway genes)  
avail_nfk = [g for g in nfkb_genes if g in adata.var_names]
sc.tl.score_genes(adata, gene_list=avail_nfk, score_name='NFkB_score',
                   use_raw=False, ctrl_size=50)

# CT39 module score
sc.tl.score_genes(adata, gene_list=ct39_genes, score_name='CT39_score',
                   use_raw=False, ctrl_size=50)
```

**Parameters:**
- `use_raw=False`: Use **log1p-normalized, highly variable genes** (not raw counts)
- `ctrl_size=50`: Control size for background gene sampling
- **Output:** New columns in `adata.obs`: `HIF1A_score`, `NFkB_score`, `CT39_score`

**Gene Sets Used:**
- **HIF1A pathway genes:** VEGFA, SLC2A1, LDHA, PGK1, ALDOA, ENO1, PDK1, ... (~15-20 genes)

### **6. "TIPPING POINT" DEFINITION**

double_high = (adata.obs['HIF1A_score'] > hif_q3) & (adata.obs['NFkB_score'] > nfk_q3)

- **Cells with simultaneous hypoxia + inflammatory stress**

**GSE241037 vs GSE274987 的核心区别**:

| 维度 | GSE241037 (CD45+ FACS) | GSE274987 (Multiome + ParseBio) |
|------|----------------------|-------------------------------|
| **细胞富集** | **CD45+ FACS 分选** → 只保留免疫细胞，没有肿瘤细胞和基质细胞 | **无预选** → 包含所有细胞类型（肿瘤、免疫、基质） |

| **区域定义** | Core / Periphery | PT (Peritumoral) / TC (Tumor Core), 排除 TE (Tumor Edge) |
| **多组学** | 仅转录组 | RNA + ATAC 配对数据 |

**对你分析的影响**:

1. **如果目标是 TAM 重编程研究** — GSE241037 的 CD45+ 分选意味着你拿到的**纯免疫细胞**，不需要再从全局数据里挑免疫亚群，但**完全缺失肿瘤细胞和 CAF 信息**（做不了 cell-cell communication 的完整分析）
2. **如果要做瘤旁 vs 肿瘤全景分析** — GSE274987 更合适，因为保留了所有细胞类型，且有 scATAC 提供表观遗传信息
3. **批次效应复杂度** — GSE274987 混合了两种完全不同的平台 (Multiome vs ParseBio)，整合难度更高

简单来说：**GSE241037 = 免疫细胞专精视角**，**GSE274987 = 全细胞类型 + 多组学视角**。
