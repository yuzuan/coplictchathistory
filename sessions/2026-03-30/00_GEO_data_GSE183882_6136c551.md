```
data[-1]               → 29545 行 × 37 列（去掉第1行）

```

所以 `data[-1] <- lapply(data[-1], ...)` 实际上是：
1. 右边：对去掉第1行的 29545 行做 `as.numeric()`，gene 列 → 全变 NA

**修复**：把 `data[-1]` 改为 `data[, -1, with = FALSE]`，或者更简单——直接转为 `data.frame` 后操作：

data <- as.data.frame(data)
data[-1] <- lapply(data[-1], as.numeric)  # data.frame 的 [-1] 是列操作
