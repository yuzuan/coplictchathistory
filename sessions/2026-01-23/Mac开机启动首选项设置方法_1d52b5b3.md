在Mac上设置开机自动启动应用程序有以下方法:

**通过系统设置:**
1. 点击左上角  菜单 → **系统设置**
2. 选择 **通用** → **登录项**
3. 点击 **+** 按钮添加想要自动启动的应用程序
4. 或者取消勾选不想自动启动的应用

**快捷方法:**
- 打开应用程序后，右键点击程序坞中的图标
- 选择 **选项** → **登录时打开**

**注意事项:**
- 某些应用程序可能会在首次启动时自动请求加入登录项
- 过多的启动项会影响Mac的启动速度
- 可以随时返回登录项设置中移除不需要的启动程序

在Mac上将微信输入法设置为首选输入法:

**设置步骤:**

1. 点击左上角  菜单 → **系统设置**

2. 选择 **键盘**

3. 点击 **输入法**（或 **文本输入** / **输入源**，根据macOS版本可能略有不同）

4. 在输入法列表中:
   - 找到**微信输入法**
   - 将其拖动到列表**最顶部**
   - 或点击编辑按钮调整顺序

5. 确保勾选了 **在菜单栏中显示输入法菜单**

**快捷切换:**
- 使用 `Control + 空格` 或 `Command + 空格` 在输入法之间切换
- 首选输入法会在切换时优先显示

**如果找不到微信输入法:**
1. 在输入法设置中点击 **+** 按钮
2. 搜索并添加 **微信输入法**
3. 然后将其移到列表顶部

设置完成后,微信输入法就会成为您的默认首选输入法。
