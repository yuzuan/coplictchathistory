# PPI 网络分析代码思路详解

这段代码使用 **STRINGdb** 和 **igraph** 包进行蛋白质-蛋白质相互作用（PPI）网络分析。以下是核心思路：

## 1. 准备差异基因列表

```r
deg_genes <- DEG_sig$gene
```
从前面差异分析中筛选出的显著差异基因作为 PPI 分析的输入。

---

## 2. 初始化 STRINGdb 连接

```r
string_db <- STRINGdb$new(version = "11.5", 
                          species = 9606,      # 9606=人类, 10090=小鼠
                          score_threshold = 400,  # 中等置信度阈值
                          input_directory = output_dir)
```

- **species**: 物种 NCBI taxonomy ID
- **score_threshold**: 互作可信度阈值（0-1000），400 = 中等置信度

---

## 3. 基因 ID 映射

```r
deg_mapped <- string_db$map(data.frame(gene = deg_genes), "gene", removeUnmappedRows = TRUE)
```

将基因符号（如 TP53）转换为 STRING 数据库的内部 ID（如 9606.ENSP00000269305）。

---

## 4. 获取 PPI 互作关系

```r
interactions <- string_db$get_interactions(deg_mapped$STRING_id)
```

从 STRING 数据库查询这些基因/蛋白之间已知的相互作用关系。

---

## 5. 构建 igraph 网络对象

```r
# STRING_id 转回基因名
id_to_gene <- setNames(deg_mapped$gene, deg_mapped$STRING_id)
interactions$from_gene <- id_to_gene[interactions$from]
interactions$to_gene <- id_to_gene[interactions$to]

# 创建无向图
g <- graph_from_data_frame(interactions_clean[, c("from_gene", "to_gene")], directed = FALSE)
```

将互作表转换为网络图对象，便于后续拓扑分析。

---

## 6. 计算 Hub 基因（中心性分析）

```r
hub_analysis <- data.frame(
    gene = V(g)$name,
    degree = degree(g),           # 度：连接的边数
    betweenness = betweenness(g), # 介数：经过该节点的最短路径数
    closeness = closeness(g),     # 接近中心性：到其他节点的平均距离
    eigenvector = eigen_centrality(g)$vector  # 特征向量中心性
)
```

**核心指标解释：**

| 指标 | 含义 | 生物学意义 |
|------|------|------------|
| **Degree** | 节点连接数 | 与多少蛋白直接互作 |
| **Betweenness** | 介数中心性 | 信息传递的"桥梁"作用 |
| **Closeness** | 接近中心性 | 影响其他节点的效率 |
| **Eigenvector** | 特征向量中心性 | 与重要节点相连的重要性 |

---

## 7. 可视化

1. **Hub 基因柱状图**：展示 Top 20 高连接度基因
2. **子网络图**：提取 Hub 基因及其邻居，高亮显示核心节点
3. **Hub 基因热图**：展示 Hub 基因在样本中的表达模式

---

## 整体流程图

```
差异基因列表
     ↓
STRING 数据库映射
     ↓
获取 PPI 互作关系
     ↓
构建 igraph 网络
     ↓
计算中心性指标 → 识别 Hub 基因
     ↓
可视化（网络图、柱状图、热图）
```

---

## 关键点

- **Hub 基因**通常是网络中连接度最高的节点，可能是关键调控因子
- **score_threshold = 400** 是中等置信度，可根据需要调整（700 = 高置信度）
- 网络分析帮助从"基因列表"上升到"功能模块"和"关键调控节点"的层面

    # 确保矩阵是数值型
    exp_hub_scaled <- as.matrix(exp_hub_scaled)
    
    # 列注释（样本分组）
    sample_group <- pd$group
    names(sample_group) <- rownames(pd)
    
    # 行注释（Up/Down）- 确保匹配顺序
    hub_change <- DEG$change[match(hub_genes_in_exp, DEG$gene)]
    names(hub_change) <- hub_genes_in_exp
    
    # 处理可能的NA值
    hub_change[is.na(hub_change)] <- "Not Sig"
    
    # 定义颜色 - 根据您的实际分组名称修改
    group_levels <- unique(pd$group)
    group_colors <- setNames(c("#4DAF4A", "#E41A1C"), group_levels)
    regulation_colors <- c("Up" = "#E41A1C", "Down" = "#377EB8", "Not Sig" = "grey60")
    
    # 顶部注释
    ha_top <- HeatmapAnnotation(
        Group = sample_group[colnames(exp_hub_scaled)],
        col = list(Group = group_colors),
        show_annotation_name = TRUE,
        annotation_name_side = "left"
    )
    
    # 行注释
    ha_row <- rowAnnotation(
        Regulation = hub_change[rownames(exp_hub_scaled)],
        col = list(Regulation = regulation_colors),
        show_annotation_name = TRUE
    )
    
    # 绘制热图
    p_hub_heatmap <- Heatmap(
        exp_hub_scaled,
        col = colorRampPalette(c("#524b9a", "white", "#e63118"))(100),
        name = "Gene Expression\n(Z-score)",
        heatmap_legend_param = list(direction = "horizontal", nrow = 1),
        show_row_names = TRUE,
        show_column_names = FALSE,
        show_row_dend = FALSE,
        show_column_dend = FALSE,
        top_annotation = ha_top,
        right_annotation = ha_row,
        column_title_side = "top",
        column_split = factor(sample_group[colnames(exp_hub_scaled)]),
        row_split = factor(hub_change[rownames(exp_hub_scaled)]),
        column_title = "Expression of Top 50 Hub Genes",
        row_names_gp = gpar(fontsize = 6),

        cluster_columns = TRUE
    )
    
    # 保存热图
    pdf(file.path(output_dir, "hub_genes_heatmap.pdf"), width = 12, height = 10)
    draw(p_hub_heatmap,
